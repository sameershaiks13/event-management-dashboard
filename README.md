# 🎉 Event Management System

A full-stack **Event Management System** designed to simplify event organization and participation within educational institutions. The application provides dedicated dashboards for students and administrators, enabling seamless event registration, approval workflows, certificate management, and user administration.

## 🚀 Features

### 👨‍🎓 Student
- User Registration & Login
- Browse Available Events
- Register for Events
- View Registered Events
- Track Registration Status
- Upload Required Documents
- Download Approved Certificates
- Manage Personal Profile

### 👨‍💼 Admin
- Secure Admin Login
- Create, Edit, and Delete Events
- Manage Student Registrations
- Approve or Reject Event Requests
- Certificate Approval & Management
- Search and Filter Students
- Dashboard with Event Statistics

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Event-Management-System.git
```

2. Navigate to the project

```bash
cd Event-Management-System
```

3. Install dependencies

```bash
npm install
```

4. Configure your MySQL database.

5. Start the server

```bash
npm start
```

---

## 📌 System Workflow

1. Student registers and logs in.
2. Student browses available events.
3. Student registers for an event.
4. Admin reviews and approves/rejects registrations.
5. Student participates in the event.
6. Admin approves certificates.
7. Student downloads the certificate.
